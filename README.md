# 塾講師シフト自動生成システム (Juku Scheduler)

## プロジェクトの概要
学習塾における1対2形式の個別指導スケジュールを自動生成するシステム。
属人的で多大な時間を要していたシフト作成業務を、局所探索アルゴリズムを用いて最適化し、制約条件を満たしたスケジュールを自動出力する。

## システム構成と技術スタック
*   **言語:** Python
*   **入力:** 講師および生徒の希望シフト、受講科目データ (CSV)
*   **出力:** 最適化された時間割データ (CSV) および スコア推移グラフ (PNG)
*   **主要ライブラリ:** `pandas`, `matplotlib`

## 採用アルゴリズムと評価関数
スケジュール生成は組み合わせ最適化問題として定式化し、以下の手法で解を探索している。

1.  **初期解の生成:** ランダムまたは貪欲法による初期シフトの割り当て。
2.  **最適化手法:** 焼きなまし法（Simulated Annealing）等による局所探索。
3.  **制約条件とペナルティ設定:**
    *   ハード制約（講師の出勤可能日、生徒の受講希望日の一致）
    *   ソフト制約（連続勤務の回避、特定の講師・生徒ペアの優先等）に対するペナルティスコアの加算。

**実行結果のハイライト:**
アルゴリズムのチューニングにより、局所最適解からの脱出を図っている。

.
├── data/                      # 入力用ベースデータ格納

│   ├── daily/                 # 日別のシフト表

│   ├── solver.py              # 実行制御
│   └── validator.py           # 制約条件チェック
├── tools/                     # 補助ツール群
│   ├── evaluate_schedule.py   # スケジュール評価ツール
│   ├── export_excel.py        # Excel出力ツール
│   ├── export_individual_files.py
│   ├── generate_mock_data.py  # テスト用データ生成
│   ├── view_schedule.py       # スケジュールプレビュー
